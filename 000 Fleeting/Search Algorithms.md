# Ai Search Algorithms
>[!definition]
>It's a concrete procedure that implements the search strategy.

- It specifies the data structure, exact steps of node execution, how the frontier is maintained.

>[!question]- How's it different from a normal search algorithm? 
>A normal search algorithm (e.g. binary search) searches within an existing data structure, whereas an AI search algorithm searches the [[State Space]] generated by a problem definition. 
>
>It's important to note that a data structure is fully known during normal search. Other the other hand, we don't store the entire space explicitly for AI search. Instead, rules describe how a state changes and the algorithm generates new states while searching.
