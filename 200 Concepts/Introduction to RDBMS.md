# RDBMS

> It's an implementation of the relational data model that provides data storage, query processing, transaction management, concurrency control and constraint enforcement, i.e, a full database engine. 

e.g. PostgreSQL, MariaDB, MySQL, Oracle DB, etc.

RDBMS implements the following things:
1. [[Logical Layer]]
2. [[Query Optimizer]]
3. [[Storage Engine]]
4. [[Transaction System]]
