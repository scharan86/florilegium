# Database Management System

> It's a software that manages a [[Introduction to Databases|db]] by providing mechanisms to **define, create, maintain and control** access to the database. 

## Responsibilities of DBMS
1. **Scheme Definition** : defines the structure of the data. 
2. **Persistent Storage**: constructs the db and stores data on persistent storage.
3. **Data Manipulation**: insert, update, delete, query.
4. Manages concurrent access from multiple users or processes.
5. **Data Protection**: access control, crash recovery, integrity constraints.
